# Build plan

## Stage 1
Create a reproducible Ubuntu-first development environment that installs:
- KDE Plasma
- SDDM
- developer tooling
- Docker
- Ollama
- shell and terminal defaults

## Stage 2
Add Kismet-specific layers:
- branding
- wallpapers
- SDDM theme
- Plymouth splash
- terminal profile
- onboarding flow

## Stage 3
Add AI integration:
- OpenClaw install/setup path
- Ollama model bootstrap
- local agent helpers
- desktop launchers and service wrappers

## Stage 4
Create installable preview paths:
- remaster script or image build flow
- recovery notes
- update and rollback path

## Notes
Zorin OS is the style benchmark for approachability and desktop polish.
Garuda Linux is a reference point for boldness and enthusiast energy, not a literal base.
Ubuntu remains the practical build foundation for now.
