# Base distribution decision

## Decision
Kismet OS is now targeting an **Ubuntu-first base**, with design and user-experience inspiration drawn from **Zorin OS** and selected performance and gamer-oriented polish ideas inspired by **Garuda Linux**.

## Why Ubuntu first
Ubuntu is the more practical starting base for Kismet OS right now because it gives:
- stable package availability
- broad hardware compatibility
- a well-known installer ecosystem
- simpler onboarding for a polished desktop-focused distribution
- easier early integration work for developer tooling and AI services

## Influence model
Kismet OS is not a clone of either project.

### Zorin-inspired elements
- polished first boot experience
- clean desktop defaults
- approachable premium-feeling UI
- cohesive branding and onboarding

### Garuda-inspired elements
- bolder visual confidence where appropriate
- performance-conscious defaults
- stronger out-of-the-box tooling attitude
- power-user friendliness without forcing raw minimalism

## Practical interpretation
This means:
- Ubuntu as the technical base
- KDE Plasma as the desktop direction
- Kismet-specific theming and integration work layered on top
- optional future experiments with an Arch branch or advanced flavor later if needed

## Current stance
Ubuntu-first is the production-minded path.
A future Arch flavor can exist later if it earns its keep.
