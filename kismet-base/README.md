# Kismet base

This directory contains the first practical starter files for building Kismet OS as a real Ubuntu-first environment.

## Current contents
- `package-manifest.ubuntu` — starter package baseline
- `bootstrap/setup-ubuntu-base.sh` — initial bootstrap script

## Intent
This is not a full ISO builder yet.
It is the first concrete layer that turns Kismet OS from a concept repo into a buildable environment plan.

## Direction
- Base: Ubuntu
- UX inspiration: Zorin OS
- attitude and power-user polish inspiration: Garuda Linux
- desktop direction: KDE Plasma
