# Kismet desktop defaults

- GNOME desktop with a Zorin-inspired bottom dock, app grid, and calm dark presentation
- GDM login flow with GNOME as the default desktop session
- dark-first presentation with rounded surfaces, soft blue accents, and low-clutter spacing
- Yaru blue theme, Yaru blue icons, Inter font defaults, and bundled Kismet wallpapers focused on polished desktop presentation
- bottom dock, pinned apps, tray, clock, and launcher flow aligned with Zorin-style usability
- zsh as the default shell
- GNOME Terminal available by default, developer-first utilities preinstalled
- Kismet AI stack centered around Hermes first, OpenCode second, local Ollama underneath
- Windows app compatibility exposed through Wine, Winetricks, and Lutris
