# Kismet desktop defaults

- KDE Plasma desktop with a Zorin-inspired bottom panel layout and a little macOS-style calm polish
- dark-first presentation with a polished, low-clutter look
- deep blue surfaces with cyan accents and softer contrast than stock Breeze
- app launcher, pinned apps, taskbar, tray, and clock aligned for familiar desktop use
- bundled Kismet wallpapers focused on AI, coding, and neural-terminal aesthetics
- zsh as the default shell
- Konsole available by default, developer-first utilities preinstalled
- Kismet AI stack centered around Hermes first, OpenCode second, local Ollama underneath
- Windows app compatibility exposed through Wine, Winetricks, and Lutris
